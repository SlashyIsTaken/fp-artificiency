# Screenshots

The main [README](../../README.md) shows two images from this folder. Capture them from a
running build (`npm run tauri dev`) and save them here with these exact names:

| File | Page | How to reach it |
|---|---|---|
| `overview.png` | Overview dashboard | The default page on launch |
| `waste.png` | Waste diagnosis | Click "Waste diagnosis" in the sidebar |

Tips:

- Frame the app window itself, not the whole desktop. On KDE, Spectacle's "Active Window"
  mode does this cleanly.
- Pick a range with data (the 30-day default is usually good), so the tiles, chart, and
  tables are populated.
- Save as PNG. A window around 1300 to 1500 pixels wide looks right, since the UI caps its
  content width.
- The app follows your system light or dark theme, so capture in whichever you prefer.
